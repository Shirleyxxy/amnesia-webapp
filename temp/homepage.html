Background

Amnesia enables users to decrementally update trained machine learning models in order to efficiently and reliably “forget” the sensitive personal data from applications that contain ML models. Our web app serves to demonstrate the decremental update procedures for an item-based collaborative filtering model. Specifically, our web app will display the corresponding changes of history matrix, cooccurrence matrix, item interactions, and item similarity matrix after adding and removing the data of a particular user.

Technical Details

  • Model: Amnesia recommender system implemented in Rust and Differential Dataflow
  • Flask: Python-based web framework
  • HTML & CSS: website structure and style
  • Apache Kafka: distributed stream processing system
  • Redis: distributed, in-memory key-value database
